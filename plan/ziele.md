## A. Ziele 2026 (jeder Punkt hat ein hartes Erfolgs-Kriterium)

- [ ] **Erstes eigenes Geld als Selbständiger verdient.** Kriterium: ≥ 1 € Einnahme mit eigener Rechnung dokumentiert.
- [ ] **1 Monat mit ≥ 5.000 € Einnahmen.** Kriterium: ein Kalendermonat mit dokumentierten Einnahmen ≥ 5.000 € (Quelle egal).
- [ ] **Social Media richtig gestartet.** Kriterium: ≥ 1.000 Follower auf Hauptplattform + ≥ 50 veröffentlichte Posts in 12 Monaten.
- [ ] **Bereit für Momentum-Launch 2027.** Kriterium: **1 Hauptprodukt komplett entwickelt + 1 Produktionsstätte vertraglich gebunden + Website live mit funktionalem Funnel + Pre-Launch-Mailing-Liste ≥ 500 Subscribers + Launch-Datum fixiert.**

- [ ] **Ausziehen von zuhause.** Kriterium: eigener Mietvertrag unterschrieben.
- [ ] **Alle BWL-Klausuren 2026 mit Note ≤ 2,0 bestanden.**
- [ ] **KI auf Top-Niveau.** Kriterium: **3 öffentliche KI-Projekte gezeigt**:
  - [ ] **Projekt A — Intern bei ZOXS** (z. B. Auswertungs-/Reporting-Automation; Andocken an Andrea-Performance-Dashboard)

  - [ ] **Projekt C — Öffentliches Demo-Projekt** (Open-Source-Tool / MCP-Server / Agent-Template, gezeigt auf GitHub + LinkedIn/Twitter). _Brand-Aufbau._
